# Project Status

The core Construction AI Estimator is fully on this repository:

- Models, config, agents (including nodes), ingestion, visualization, utils
- Prompts, docs, Tauri 2 shell, Python sidecar, UI frontend
- CLI entry point (`run_analysis.py`)
- Build scripts for sidecar and desktop packaging

Optional full desktop (Flet) and Streamlit UI source files that were very large were summarized or partially included. The Tauri + Python sidecar path is the primary recommended desktop application.

Clone and start with the README and `docs/TAURI_SETUP.md`.
